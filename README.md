# Japanese Walking Timer

**ジャパニーズウォーキング（高強度インターバル歩行）** のためのインターバル速歩タイマーです。3分速歩と3分ゆっくり歩行を繰り返し、5セット（約30分）のトレーニングをサポートします。

---

## ジャパニーズウォーキングとは

近年 SNS で「ジャパニーズウォーキング」として話題になっているのは、**信州大学の能勢裕医師らが提案した高強度インターバル歩行（Interval Walking Training：IWT）** です。

### 基本的なやり方

| フェーズ | 時間 | 強度の目安 |
|---------|------|------------|
| **速歩** | 3分 | VO₂peak の約 70％。会話ができないほどのペース |
| **ゆっくり歩行** | 3分 | VO₂peak の約 40％まで落として体力を回復 |

- この **6分間を1セット** とし、**5セット以上（30分程度）** 繰り返す
- 推奨頻度は **週4〜5回**

従来の連続歩行より運動強度を高くしつつ、全体の運動量は抑えられるため、忙しい人や高齢者でも続けやすいのが特徴です。

## 科学的エビデンス

### 運動能力・健康指標の向上

- **有酸素能力（VO₂peak）の改善**  
  信州大学の増木しずえらによる研究（2019年、65歳前後 679名、5か月間）では、週50分以上の IWT で VO₂peak が平均 **14％向上** し、生活習慣病のリスクが **17％減少**。アメリカ心臓協会が推奨する75分／週の高強度運動より少ない時間で同等以上の効果が報告されています。ガーディアン紙（2025年）では、IWT が連続歩行より **15〜20％大きなフィットネス向上** をもたらすと紹介されています。

- **血圧・筋力の改善**  
  日本の研究（平均63歳、200名以上）では、IWT により血圧の低下・大腿筋力の向上・VO₂max の改善が示されています。通常のウォーキングでは得られにくい身体への刺激があり、体力が停滞している人に有効と医師がコメントしています（Verywell Health 等）。

- **骨密度の改善**  
  PLOS One（2024年）では、平均64歳の閉経後女性234人が5か月間、1日5セット以上・週4日以上の IWT を実施した結果、初期骨密度の低い人では腰椎・大腿骨頸部の骨密度が **1.0〜1.8％増加**。重篤な副作用はほとんどなかったと報告されています。

- **糖代謝への効果**  
  デンマークの Karstoft らによる無作為化比較試験では、2型糖尿病患者を対象に IWT を行ったところ、同じエネルギー消費量の連続歩行と比較して **血糖コントロール（HbA1c）と体組成がより改善**。別の無作為化試験では、IWT は連続歩行よりフィットネス向上が大きく、**体重が平均3〜5kg減少** したと紹介されています。

## このアプリの使い方

1. **index.html** をブラウザで開く（ローカルまたは Web サーバーで配信）
2. 「再生」ボタンでタイマーを開始
3. **3分速歩** → **3分ゆっくり歩行** が自動で切り替わります
4. 5セット（約30分）完了まで、表示に従って歩行を続けてください

- 一時停止・リセットも可能です
- 音声通知（Tone.js）でフェーズの切り替えを知らせます（対応環境の場合）

### 表示の見方

- **Set X / 5** … 全5セットのうち、現在何セット目かを表示
- **ドット（●○）** … 1セット内の2フェーズ（速歩・ゆっくり歩行）の進捗を表示
  - ○○ … 準備完了
  - ●○ … 速歩中（1/2）
  - ●● … ゆっくり歩行中（2/2）または全セット完了
